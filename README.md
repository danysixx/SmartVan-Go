# SmartVan-Go
SmartVan-Go es una plataforma de movilidad universitaria inteligente que optimiza el transporte dentro del campus. Integra tecnologías en la nube, analítica de datos e inteligencia artificial para mejorar la disponibilidad de vehículos, reducir tiempos de espera y ofrecer información en tiempo real sobre rutas y horarios.

El proyecto busca ser un modelo replicable de transporte sostenible y eficiente, fomentando innovación, colaboración y conciencia ambiental en la comunidad universitaria.
